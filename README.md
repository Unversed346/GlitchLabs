GlitchLabs is a python app the allows you to corrupt audio and video files. It does this by editing Hex data!

---

# Installation:

## We will be using install.sh

### Clone the repo
```bash
git clone https://github.com/Unversed346/GlitchLabs.git
```
### Enter the application directory
```bash
cd GlitchLabs
```
### Make install.sh executable
```bash
chmod +x install.sh
```
### Run install.sh
```bash
./install.sh
```
